# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] - 2026-06-29

### Added

- 本地媒体文件扫描与 blake3 内容哈希索引
- 标签与文件内容绑定，支持移动/重命名后标签跟随
- 搜索、多维筛选、文件夹树、虚拟滚动列表
- 全局标签管理、批量打标、标签导入/导出（JSON）
- 图片全屏预览、视频全屏播放
- 浅色主题与可配置 CSS 变量主题系统
